# Setup Instructions for Varkos-store

## Prerequisites
1. **Node.js** - Ensure you have Node.js installed. You can download it from [nodejs.org](https://nodejs.org/).
2. **npm** - npm is included with Node.js. Verify installation by running `npm -v` in your terminal.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Varkos-source/Varkos-store.git
   cd Varkos-store
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables: 
   - Create a `.env` file in the root directory and add the required environment variables.
   - You may refer to `.env.example` for reference.

4. Run the application:
   ```bash
   npm start
   ```

## Additional Configuration
- Refer to the [documentation](https://github.com/Varkos-source/Varkos-store/wiki) for more advanced configuration options.
